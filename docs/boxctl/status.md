# boxctl status

Show system health: RAUC slot info, AppArmor, dm-verity, failed units, disk and memory usage.

## Usage

```
Usage: boxctl status

Show system health: RAUC slot info, AppArmor, dm-verity, failed units,
disk and memory usage.
```

## Library modules

Uses: [`interact`](../interact.md), [`rauc`](../rauc.md), [`health`](../health.md), [`system`](../system.md)
