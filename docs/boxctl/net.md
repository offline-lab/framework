# boxctl net

## Usage

```
Usage: boxctl net <subcommand> [args]

Subcommands:
  status                Show current WiFi state and IP
  scan                  Scan for available networks
  connect <ssid> <psk>  Connect to a WiFi network
```

## Library modules

Uses: [`interact`](../interact.md), [`wifi`](../wifi.md), [`system`](../system.md)
